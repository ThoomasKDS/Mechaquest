\doxysubsection{Description détaillée}
Représente l’inventaire d’un joueur contenant les objets utilisables. 

Cette structure stocke la quantité de chaque type d’objet que le joueur possède. Les objets sont utilisés principalement pendant les combats ou lors des déplacements.
