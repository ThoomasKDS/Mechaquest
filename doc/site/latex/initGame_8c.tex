{\footnotesize\ttfamily void clean\+Up (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Libère proprement toutes les ressources utilisées par le jeu avant de quitter. 

Cette fonction détruit les textures des arrière-\/plans, le renderer, la fenêtre, la police chargée, et quitte correctement les modules SDL (SDL\+\_\+ttf, SDL\+\_\+image, SDL).


\begin{DoxyParams}{Paramètres}
{\em game} & Pointeur vers une structure {\ttfamily \doxylink{structgame__t}{game\+\_\+t}} contenant toutes les ressources à libérer.\\
\hline
\end{DoxyParams}
\begin{DoxyPrecond}{Précondition}
Le pointeur {\ttfamily game} doit être valide et initialisé, ainsi que toutes les ressources associées (textures, renderer, fenêtre, police). 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
Toutes les ressources allouées dynamiquement par SDL sont libérées, et SDL est correctement quitté.
\end{DoxyPostcond}
\begin{DoxyNote}{Note}
Cette fonction doit être appelée systématiquement avant la fermeture de l\textquotesingle{}application pour éviter les fuites mémoire. 
\end{DoxyNote}

{\footnotesize\ttfamily int init\+\_\+game (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Initialise l\textquotesingle{}environnement du jeu en préparant SDL et ses modules associés. 

Cette fonction initialise SDL, SDL\+\_\+ttf, SDL\+\_\+image et SDL\+\_\+mixer pour gérer l\textquotesingle{}affichage graphique, les polices, les images et le son. Elle crée une fenêtre principale en mode plein écran ainsi qu\textquotesingle{}un renderer pour gérer l\textquotesingle{}affichage. Une police spécifique est également chargée pour le rendu des textes dans le jeu.


\begin{DoxyParams}{Paramètres}
{\em game} & Pointeur vers une structure {\ttfamily \doxylink{structgame__t}{game\+\_\+t}} où seront stockées les ressources créées (fenêtre, renderer, police).\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Renvoie}
int
\begin{DoxyItemize}
\item {\ttfamily 1} si l\textquotesingle{}initialisation complète réussit.
\item {\ttfamily 0} en cas d\textquotesingle{}erreur dans l\textquotesingle{}initialisation de SDL, SDL\+\_\+image, SDL\+\_\+mixer, ou à la création de la fenêtre/renderer.
\item {\ttfamily 1} si l\textquotesingle{}erreur provient de SDL\+\_\+ttf (ce cas particulier peut être une incohérence et devrait être vérifié dans l\textquotesingle{}implémentation).
\end{DoxyItemize}
\end{DoxyReturn}
\begin{DoxyPrecond}{Précondition}
Le pointeur {\ttfamily game} doit être valide et non NULL. 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
La structure {\ttfamily game} est remplie avec les ressources initialisées. En cas d\textquotesingle{}erreur, toutes les ressources allouées précédemment sont libérées.
\end{DoxyPostcond}
\begin{DoxyNote}{Note}
La fonction initialise également la génération aléatoire ({\ttfamily rand()}).
\end{DoxyNote}
\begin{DoxyWarning}{Avertissement}
Le chemin vers le fichier de police {\ttfamily "{}img/police\+\_\+temporaire.\+ttf"{}} doit être valide, sinon l\textquotesingle{}initialisation échoue. 
\end{DoxyWarning}
