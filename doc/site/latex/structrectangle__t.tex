\doxysubsection{Description détaillée}
Structure représentant un rectangle graphique avec du texte à afficher dans une interface SDL. 

Cette structure permet de définir facilement un élément graphique composé d\textquotesingle{}un rectangle coloré, accompagné éventuellement d\textquotesingle{}un texte centré ou positionné à l\textquotesingle{}intérieur. Elle est principalement utilisée pour les boutons, les menus ou les dialogues dans une interface graphique.
