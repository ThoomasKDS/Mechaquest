Description détaillée

Structure regroupant les textures utilisées pour afficher les PNJs du jeu.

Cette structure centralise les textures représentant les personnages non-joueurs (PNJs) dans différentes orientations et pour certains PNJs spécifiques :

Note
Chaque texture doit être chargée avant utilisation via une fonction dédiée (charger_texture) depuis les fichiers d'images correspondants.
Avertissement
Il est impératif de libérer explicitement chacune de ces textures (via SDL_DestroyTexture) lors de la fermeture du jeu afin d'éviter toute fuite mémoire.
